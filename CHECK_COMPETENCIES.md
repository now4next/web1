# 🔍 역량 키워드 확인하기

## 로그 분석 결과

Console 로그를 보니 다음 정보가 보입니다:
```
competencies: Array(12)
```

12개의 역량이 있는데, 구체적인 키워드를 확인해야 합니다.

---

## 즉시 확인 방법

### Console에 다음 명령어를 복사해서 붙여넣으세요:

```javascript
console.log('=== 선택된 역량 키워드 ===')
console.log(assessmentQuestions.map(q => q.competency))
```

**Enter 키를 누르면 다음과 같이 표시됩니다:**
```javascript
["역량1", "역량2", "역량3", ...]
```

---

## 이 키워드를 저에게 알려주세요!

예시:
```
["리더십", "문제해결", "협업", ...]
```

---

## 또는 Console 스크린샷

1. F12 → Console 탭
2. 위 명령어 실행
3. 결과 전체 스크린샷
4. 공유

---

**이 정보만 있으면 즉시 해결할 수 있습니다!** 🚀
